**Программа:** Notion (веб-версия и десктоп-приложение для Windows)

**Источник:** notion.com (официальный сайт)

**Ключевые функции:**
- Создание и редактирование страниц и баз данных
- Notion AI и Custom Agents (автоматизация задач) 
- Совместная работа и комментарии
- Импорт/экспорт данных
- Офлайн-режим
- People Directory и Jira-интеграция

## 2. План тестирования

### 2.1 Цели тестирования
- Проверить корректность функциональности: создание страниц, баз данных, работу AI-агентов, совместную работу.
- Оценить соответствие GUI обновлённому дизайну (спейсинг, отступы, тёмная тема, адаптивность) .
- Выявить уязвимости безопасности (авторизация, шифрование, сессии, доступ к данным) .

### 2.2 Объём тестирования

**Функциональное тестирование:**
- Регистрация / вход / выход
- CRUD страниц и баз данных
- Работа с блоками (/команды)
- Notion AI и Custom Agents
- Совместная работа (комментарии, @упоминания)
- Импорт/экспорт (Markdown, CSV, PDF)
- Офлайн-режим
- People Directory и интеграции 

**Тестирование GUI :**
- Соответствие новой системе отступов и паддингов (обновление дизайна март 2026) 
- Тёмная и светлая темы
- Адаптивность под разные разрешения экрана
- Локализация (русский, английский, японский)
- Шрифты, иконки, анимации

**Тестирование безопасности :**
- Проверка авторизации и управления доступом
- Анализ шифрования (HTTPS, HSTS, SSL/TLS) 
- Проверка сессий и cookie (HttpOnly, Secure) 
- Аудит логов и экспорт данных 
- Проверка уязвимостей (CVE-2026-25020 не относится к основному Notion, но учитываем плагины) 
- SIEM/DLP интеграции для Enterprise 

### 2.3 Инструменты
- **ОС:** Windows 11 Pro
- **Браузеры:** Chrome 134, Firefox 135, Edge 134
- **Программное обеспечение:**
  - Charles Proxy / Wireshark (анализ трафика)
  - Burp Suite Community (безопасность)
  - Notion API (для тестирования интеграций) 
  - Developer Tools (Chrome DevTools)
  - UpGuard / SSL Labs (оценка безопасности) 

### 2.4 Критерии качества
- **Критичные дефекты:** отсутствуют (приложение не крашится, данные не теряются, нет утечек).
- **Основные дефекты:** не более 3-х.
- **Незначительные дефекты:** не более 7-ми.
- **Прохождение тест-кейсов:** 100% для критических, ≥90% для остальных.

### 2.5 Расписание
- Планирование и подготовка: 2 часа
- Проведение тестов: 5 часов
- Составление отчёта: 2 часа
